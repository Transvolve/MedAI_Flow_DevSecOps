# ISO/IEC 27001 ISMS Controls Mapping

- Access Control: JWT/Bearer tokens; RBAC (future); least privilege in CI/CD.
- Asset Management: SBOM for images; dependency pinning (future).
- Cryptography: TLS in transit; no PHI at rest by default.
- Operations Security: Container scanning; image signing (future); audit logs.
- Supplier Relationships: GitHub/Azure; documented SLAs.
